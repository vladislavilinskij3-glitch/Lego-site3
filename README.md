# Лего-Студия MIRACLE (г. Вологда)
Одностраничный сайт (лендинг) для Лего-студии, созданный в рамках итоговой аттестации по веб-разработке. В проекте картинки подгружаются с сервера **(https://api.giphy.com/v1/gifs/search?api_key=${API_KEY}&q=lego+robot&limit=7&rating=g)**
*Количество запросов в час ограничено, поэтому при ошибке gif подгружаются из локальных папок
## Ссылка на сайт
Проект опубликован на GitHub Pages и доступен по ссылке:  
**[https://vladislavilinskij3-glitch.github.io/Lego-site3/)**

## 🛠 Используемые технологии
*   **HTML5** — семантическая верстка.
*   **CSS3** — Flexbox/Grid для адаптивной сетки, переменные для смены темы.
*   **JavaScript** — интерактивные элементы (модальные окна, слайдер с картинками(/gif), переключение темы, регистрация пользователей).
*   **Git & GitHub Pages** — система контроля версий и хостинг.
*   **ВКонтакте API** — интеграция виджетов сообщества (возможность подписаться на сообщество при наведении на "ВК ГРУППА").

## ✨ Реализованный функционал
*   **Адаптивная верстка:** Сайт корректно отображается на ПК, планшетах и мобильных устройствах.
*   **Слайдер:** В каждой карточке услуг реализован слайдер для просмотра нескольких изображений.
*   **Модальные окна:** Окна для входа, регистрации, записи на пробное занятие, обратной связи и увеличения изображений.
*   **Регистрация/Вход:** Сохранение данных пользователя в `localStorage` с выводом приветствия после входа.
*   **Переключение темы:** Возможность смены светлой/темной темы с сохранением выбора в `localStorage`.
*   **Валидация форм:** Формы «Записаться на пробное» и «Написать нам» проходят проверку на заполнение и корректность данных, выводят результат в `console.log`.
*   **Интеграция с ВК:** Выпадающая карточка сообщества ВК при наведении в правом верхнем углу.
